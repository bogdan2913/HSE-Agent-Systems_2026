# Skill: Формирование аналитического отчёта

## Требуемые инструменты
- list_analyses (локальный)
- read_analysis (локальный)
- save_report (локальный)

## Когда активировать
Этот skill активируется, когда:
- Анализ промптов (навык `prompt_analysis`) завершён и нужно агрегировать результаты
- Пользователь просит сформировать сводный отчёт / итог по всем анализам
- Нужно выявить общие паттерны и лучшие практики

## Протокол действий

### Шаг 1. Получи список всех анализов
Вызови `list_analyses()`. Если анализов нет — сообщи об этом и завершись.
Запомни все доступные имена файлов.

### Шаг 2. Прочитай каждый анализ
Для каждого имени из Шага 1 вызови `read_analysis(name)`.
Из каждого анализа извлеки:
- Итоговую оценку (число из строки "Итоговая оценка: X/15")
- Оценки по каждому из 5 критериев
- Сильные стороны (список)
- Слабые стороны (список)
- Ключевые паттерны

### Шаг 3. Агрегируй данные

Вычисли для каждого критерия:
- Среднюю оценку по всем промптам
- Минимальную и максимальную оценку

Определи:
- Топ-3 промпта по итоговой оценке
- Самый часто встречающийся паттерн (по ключевым паттернам)
- Самую распространённую слабую сторону

### Шаг 4. Сформируй отчёт строго по шаблону

```
# Аналитический отчёт по системным промптам

## Общая статистика
- Проанализировано промптов: {N}
- Средняя оценка: {X.X}/15
- Диапазон оценок: от {min} до {max}

## Матрица критериев

| Промпт | Role/Persona | Few-shot | CoT | Output Format | Guardrails | Итого |
|---|---|---|---|---|---|---|
| {имя 1} | {оценка} | {оценка} | {оценка} | {оценка} | {оценка} | {сумма}/15 |
| {имя 2} | ... | ... | ... | ... | ... | .../15 |
...

## Средние оценки по критериям

| Критерий | Средняя оценка | Вывод |
|---|---|---|
| Role / Persona | {X.X} | {слабо / удовлетворительно / хорошо} |
| Few-shot Examples | {X.X} | ... |
| Chain-of-Thought | {X.X} | ... |
| Output Format | {X.X} | ... |
| Guardrails / Constraints | {X.X} | ... |

## Топ-3 промпта

1. **{имя}** — {X}/15 — {1 предложение почему}
2. **{имя}** — {X}/15 — {1 предложение почему}
3. **{имя}** — {X}/15 — {1 предложение почему}

## Общие паттерны (встречаются в большинстве промптов)
- {паттерн 1}
- {паттерн 2}
- {паттерн 3}

## Распространённые слабые стороны
- {слабость 1} — встречается в {N} из {всего} промптов
- {слабость 2} — ...

## Лучшие практики (из топ-промптов)
- {практика 1}
- {практика 2}
- {практика 3}

## Рекомендации для нового промпта
- {рекомендация 1}
- {рекомендация 2}
- {рекомендация 3}
```

### Шаг 5. Сохрани отчёт

Вызови `save_report(name="analysis_report", content=...)` с полным текстом отчёта из Шага 4.
После сохранения сообщи: "Отчёт сохранён. Готов к генерации промпта."
